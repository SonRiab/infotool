Information Portal
==================

 This software is the result of an university course.  

Requirements
------------

 - Python 2.7
 - Django (1.6.5) - `pip install django` 
 - Django-TinyMCE (1.5.2) - `pip install django-tinymce`
 - pytz - `pip install pytz`
